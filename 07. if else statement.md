## Control Flow Fundamentals (JavaScript)

### 🔹 if

Runs when:

1. Its condition is true

---

### 🔹 else if

Runs when:

1. ALL previous conditions are false  
   AND
2. Its own condition is true

---

### 🔹 else

Runs when:

1. ALL previous conditions are false

---

### 🔹 Multiple independent if blocks

- Each `if` is evaluated independently
- All conditions are checked
- Multiple blocks **can run**

Example:

```js
if (cond1) {
  /* runs if cond1 true */
}
if (cond2) {
  /* runs if cond2 true */
}
```

---

### 🔹 Nested if blocks

Runs when:

1. The outer condition is true
2. Then the inner condition is evaluated

Example:

```js
if (cond1) {
  if (cond2) {
    // runs only if BOTH cond1 and cond2 are true
  }
}
```

---

## 🧠 Mental Models

- `if / else if / else` → **choose ONE path**
- Multiple `if` → **check ALL paths**
- Nested `if` → **condition inside a chosen path**
