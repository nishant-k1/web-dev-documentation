# Types in TypeScript

TypeScript offers a range of types to ensure code reliability and type-checking. Here's an overview:

1. **Primitive Types**: Basic types like `string`, `number`, `boolean`, `null`, `undefined`, `symbol`.

2. **Object Types**: Describes objects, e.g., `{ name: string, age: number }`.

3. **Array Types**: For arrays, like `number[]` or `Array<number>`.

4. **Tuple Types**: Fixed-length arrays, e.g., `[string, number]`.

5. **Enum Types**: Grouping related constants, e.g., `enum Color { Red, Green, Blue }`.

6. **Union Types**: Combining types, e.g., `string | number`.

7. **Intersection Types**: Merging types, e.g., `TypeA & TypeB`.

8. **Function Types**: Function shapes, e.g., `(arg1: string) => void`.

9. **Generics**: Reusable types, e.g., `Array<T>`.

10. **Utility Types**: Common transformations, e.g., `Partial<T>` or `Readonly<T>`.
