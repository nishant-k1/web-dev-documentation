# ⚠️ Edge Cases in React Forms with FormData (File Uploads)

---

## 1. Empty / Undefined Values

### Problem

```js
data.append("name", undefined);
```

👉 Gets sent as `"undefined"` (string)

### Fix

```js
if (value !== undefined && value !== null) {
  data.append(key, value);
}
```

---

## 2. File Not Selected

### Problem

User submits form without selecting a file

### Fix (Frontend)

```js
if (file) {
  data.append("file", file);
}
```

### Fix (Backend)

Handle file as optional unless required

---

## 3. Multiple Files

### Problem

```html
<input type="file" multiple />
```

### Fix

```js
Array.from(files).forEach((file) => {
  data.append("files", file);
});
```

---

## 4. Arrays (Checkbox / Multi-select)

### Problem

```js
hobbies: ["cricket", "music"];
```

### Correct

```js
hobbies.forEach((hobby) => {
  data.append("hobbies", hobby);
});
```

---

## 5. Nested Objects

### Problem

```js
address: { city: "Bangalore", pin: 560001 }
```

### Fix

```js
data.append("address", JSON.stringify(address));
```

---

## 6. Boolean Values

### Problem

```js
data.append("isActive", false);
```

👉 Sent as string

### Fix

```js
data.append("isActive", String(value));
```

---

## 7. Number Values

👉 Sent as string, convert in backend if needed

---

## 8. Content-Type Header Mistake

### Wrong

```js
headers: { "Content-Type": "multipart/form-data" }
```

### Correct

Let browser handle it automatically

---

## 9. Large File Uploads

### Fix

```js
onUploadProgress: (progressEvent) => {
  console.log(progressEvent.loaded / progressEvent.total);
};
```

---

## 10. File Type Validation

```js
if (!file.type.startsWith("image/")) {
  alert("Only images allowed");
}
```

---

## 11. File Size Validation

```js
if (file.size > 2 * 1024 * 1024) {
  alert("Max 2MB allowed");
}
```

---

## 12. Same File Re-upload Issue

```js
e.target.value = "";
```

---

## 13. Backend Field Name Mismatch

Ensure names match exactly

---

## 14. Mixing JSON + FormData

### Wrong

```js
JSON.stringify(data);
```

### Correct

```js
body: data;
```

---

## 15. Debugging FormData

```js
for (let [key, value] of data.entries()) {
  console.log(key, value);
}
```

---

# 🎯 Mental Model

FormData is:

- Flat
- String-based (except File)
- Manually constructed
