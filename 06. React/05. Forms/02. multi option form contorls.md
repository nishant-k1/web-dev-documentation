# Multi Option Form Controls

## 1. checkbox

retrieve values using `event.target.checked`

## 2. select (multiple)

retrieve values using `event.target.selectedOptions`

## 2. files

retrieve values using `event.target.files`

## Conclusion

**Only 3 real multi-value cases:**

1. checkbox -> you manage
2. multi-select -> browser provides `selectedOptions`
3. file input -> browser provides `files`

The common multi-value form controls are checkbox groups, multi-ways to access values are checked + value (manual handling) for checkboxes, e.target.selectedOptions for multi-select, and e.target.files for file inputs.
