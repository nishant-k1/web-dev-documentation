# op 20 React JS Interview Questions, These are the most frequently asked React questions in interviews at top companies:

## Core Concepts & Fundamentals

1. What is the Virtual DOM and how does React's reconciliation algorithm work?
2. What is the difference between Controlled and Uncontrolled components?
3. Explain the React component lifecycle (class vs functional with hooks).
4. What is JSX? How does the browser understand it?
5. What are keys in React and why are they important in lists?
6. Hooks & State Management
7. Explain useState, useEffect, useRef, useCallback, and useMemo when do you use each?
8. What are the rules of Hooks and why do they exist?
9. How does useContext work and what problem does it solve?
10. What is the difference between useCallback and useMemo?
11. How do you create a custom Hook? Give a real-world example.

## Performance & Optimization

1. How do you prevent unnecessary re-renders in React? (React.memo, useMemo, useCallback)
2. What is code splitting and lazy loading in React?
3. What is React Suspense and how does it handle async data fetching?
4. Explain the Fiber architecture and how it improves rendering performance.

## Advanced Patterns & Architecture

1. What are Higher-Order Components (HOC) and Render Props patterns? When do you use them?
2. How does Redux work? Explain actions, reducers, and the store. How is it different from Context?
3. What is React's Error Boundary and how do you implement it?
4. Explain Server-Side Rendering (SSR) vs Client-Side Rendering (CSR) in React apps.
5. What are Portals in React and when would you use them?
6. How do you approach testing React components? (unit, integration, RTL)?
