# List Optimization (Virtualized Lists)

List rendering can be very expensive when:

- The list has hundreds or thousands of items
- Each item is a complex React component
- The list re-renders frequently

---

## Core Idea

Instead of rendering all items at once:

- Render only the items that are visible in the viewport
- Reuse DOM nodes as the user scrolls
- This technique is called **virtualization**

---

## Popular Libraries

- `react-window`
- `react-virtualized`
- `react-virtual`

---

## When to Mention in Interview

> “For very long lists, I use list virtualization (e.g., react-window) so that only visible rows are rendered, which significantly reduces DOM nodes and improves scroll performance.”
