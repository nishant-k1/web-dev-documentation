## Inline Caching

```javascript
const findUser = () => `found ${user.firsName} ${user.lastName}`;
const userData = {
  firstName: "Johnson",
  lastName: "Junior",
};

findUser(userData); // due to inline caching (optimization technique of the compiler) it can get optimized after repeated calls
```

## Hidden Classes

```javascript
const Animal = (x, y) => {
  this.x = x;
  this.y = y;
};

const obj1 = new Animal(1, 2);
const obj2 = new Animal(3, 4);

obj1.a = 30;
obj1.b = 100;
obj2.b = 30;
obj2.a = 100;
```

Conclusion:

- We should write code which is predictable not just for humans but for machines as well.
- Our code should follow DRY principles.
