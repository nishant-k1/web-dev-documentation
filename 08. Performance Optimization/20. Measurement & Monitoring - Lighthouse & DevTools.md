# Measurement & Monitoring – Lighthouse & DevTools

These tools help you **measure** and **debug** performance issues.

---

## Lighthouse

- Runs lab performance audits.
- Reports scores for:
  - Performance
  - Accessibility
  - Best Practices
  - SEO
- Shows opportunities:
  - Reduce bundle size
  - Optimize images
  - Remove render-blocking resources

---

## Chrome DevTools – Network Tab

- Inspect:
  - Request waterfall
  - Blocking vs download vs TTFB
  - Caching headers
- Helps identify:
  - Slow endpoints
  - Unnecessary requests

---

## React DevTools Profiler

- Shows which components re-render
- Measures render time per component
- Helps find:
  - Unnecessary re-renders
  - Heavy components needing memoization or splitting

---

## Interview Line

> “I use Lighthouse for high-level audits, the Network tab to debug request timing and caching, and React DevTools Profiler to track down unnecessary re-renders and heavy components.”
