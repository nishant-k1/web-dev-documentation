# Build-Time – Dependency Optimization

Goal: **Remove large/unnecessary dependencies and use lighter alternatives**.

---

## Tactics

- Audit bundle (via bundle analyzer)
- Remove unused libraries
- Replace heavy libs with lighter ones:
  - `lodash` → `lodash-es` or single-function imports
  - Moment.js → date-fns / Day.js
- Prefer native browser APIs where possible

---

## Interview Line

> “I regularly analyze the bundle, remove unused or heavy dependencies, and replace them with lighter alternatives to keep the initial JavaScript payload small.”
