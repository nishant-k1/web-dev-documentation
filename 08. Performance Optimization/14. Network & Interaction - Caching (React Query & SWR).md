# Network & Interaction – Caching (React Query & SWR)

Caching avoids **re-fetching the same data** and improves perceived performance.

---

## Libraries

- **React Query**
- **SWR**

Both provide:

- Request deduplication
- Background refetch
- Stale-while-revalidate patterns

---

## Benefits

- Fewer network calls
- Faster subsequent views
- Better offline / poor-network behavior

---

## Interview Line

> “For network performance, I use React Query/SWR to cache API responses, deduplicate requests, and refresh data in the background, which makes the UI feel much faster.”
