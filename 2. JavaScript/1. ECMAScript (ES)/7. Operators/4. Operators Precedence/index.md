# Operators Precedence

| Operator Type  | Example     | Precedence |
| -------------- | ----------- | ---------- |
| **Arithmetic** | `* / % + -` | **High**   |
| **Comparison** | `> < ==`    | Medium     |
| **Logical**    | `&&`        | **Low**    |
| **Assignment** | `=`         | Very Low   |
