# Ternary Operator

Only one ternary operator exists in JS:

```js
condition ? valueIfTrue : valueIfFalse;
```

| Type    | Arity      | Examples          |
| ------- | ---------- | ----------------- |
| Unary   | 1 operand  | `!x`, `typeof x`  |
| Binary  | 2 operands | `a + b`, `a && b` |
| Ternary | 3 parts    | `cond ? x : y`    |
