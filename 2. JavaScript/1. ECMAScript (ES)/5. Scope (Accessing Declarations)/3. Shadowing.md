# Shadowing

- Shadowing in JavaScript occurs when a variable declared in an inner scope has the same name as a variable declared in an outer scope.

- When you have nested scopes (like functions within functions), and you declare a variable with the same name in both the inner and outer scopes, the variable in the inner scope "shadows" the variable in the outer scope.

- Within the inner scope, the interpreter will use the value of the inner variable, effectively "hiding" the value of the outer variable.
  ![alt text](image.png)

- Shadowing allows you to use the same variable name in different scopes, but it can lead to confusion if not used thoughtfully.

Shadowing means:  
**A variable declared in an inner scope has the same name as one in the outer scope, thereby hiding (“shadowing”) the outer one.**

## 1. Basic Shadowing

```js
let x = 10;

function test() {
  let x = 20; // shadows outer x
  console.log(x);
}

test(); // ?
console.log(x);
```

---

## 2. Shadowing in Block Scope

```js
let value = "outer";

{
  let value = "inner"; // shadows
  console.log(value);
}

console.log(value);
```

---

## 3. Shadowing With var vs let

```js
var a = 1;

{
  let a = 2; // shadows, but different scope type
  console.log(a);
}

console.log(a);
```

---

## 4. Shadowing With Function Parameters

```js
let num = 5;

function demo(num) {
  // parameter shadows outer num
  console.log(num);
}

demo(10);
console.log(num);
```

---

## 5. Shadowing in Nested Functions

```js
let x = "global";

function outer() {
  let x = "outer";

  function inner() {
    let x = "inner"; // shadows outer's x and global x
    console.log(x);
  }

  inner();
  console.log(x);
}

outer();
console.log(x);
```

---

## 6. Shadowing + Hoisting Trap

```js
let x = "outer";

function test() {
  console.log(x); // ?
  var x = "inner"; // var hoisted => x is undefined above
}

test();
```

This is **shadowing + hoisting**, causing confusion.

---

## 7. Shadowing With const

```js
const theme = "light";

function toggle() {
  const theme = "dark"; // shadows outer theme
  console.log(theme);
}

toggle();
console.log(theme);
```
