# ES12 (ECMAScript 2021) features

- **Temporal API: A new API for working with dates and times, providing better precision and flexibility.**
- **Numeric Separator: Underscores can be used to separate digits in numbers for improved readability.**
- **Promise.any: Returns the first fulfilled promise from an iterable, rejecting if all promises are rejected.**
- **AggregateError: A new error type for grouping multiple errors into a single error.**
- **ReplaceAll method: Replaces all occurrences of a pattern in a string.**
- **Intl.partitionNumber: Partitions a number into groups based on locale-specific rules.**
- **Intl.relativeTimeFormat: Formats relative time differences (e.g., "2 days ago").**
- **WeakRef API: Creates weak references to objects, allowing for garbage collection.**
- **FinalizationRegistry: Provides callbacks when objects are garbage collected.**
