# ES7 Features

- **Async/Await**
- **Exponentiation Operator**
- **Array Includes Method**
- **PadStart and PadEnd Methods**
- **Object.entries, Object.values, and Object.keys Methods**
- **Function.prototype.toString Method**
- **Array.prototype.fill Method**
- **Array.prototype.find and Array.prototype.findIndex Methods**
- **String.prototype.startsWith and String.prototype.endsWith Methods**
- **BigInt**
