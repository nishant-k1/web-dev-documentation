# ES13 (ECMAScript 2022) features

- **Class Fields: Simplified syntax for defining class fields, including static fields and private fields.**
- **Top-Level Await: Allows for top-level await expressions in modules.**
- **Array.prototype.at: Retrieves an element from an array by its index, supporting negative indices.**
- **Object.hasOwn: Determines if an object has its own property.**
- **String.prototype.replaceAll: Replaces all occurrences of a pattern in a string, regardless of regular expression flags.**
- **Intl.NumberFormat: Improved formatting for currency and other numbers.**
- **WebAssembly SIMD: Support for Single Instruction Multiple Data (SIMD) operations in WebAssembly.**
- **Temporal API: Additional features and improvements to the Temporal API for working with dates and times.**
