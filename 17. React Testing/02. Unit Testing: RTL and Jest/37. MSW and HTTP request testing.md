# MSW and HTTP request Testing

## What is mock/ mocking

It means to create a dummy for someting

## Why don't test with real data in stead of mock data

3rd party APIs requires money for every call
API testing is complex sometimes

## What is MSW

We test api calling using MSW
Mostly we test REST api by mocking
MSW stands for `mock service worker`
React testing library recommend to use MSW
We can mock api with jest also
