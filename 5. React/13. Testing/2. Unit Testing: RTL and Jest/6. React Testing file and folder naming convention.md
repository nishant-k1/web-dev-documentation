# File and Folder naming convention

## What file name we can use for test case file?

`file_name.test.js`
`file_name.test.jsx`
`file_name.test.ts`
`file_name.test.tsx`

OR,

`file_name.spec.jsx`
`file_name.spec.js`
`file_name.spec.ts`
`file_name.spec.tsx`

## What folder name we can use for test files?

`__tests__`

Inside `__tests__` files all the files will be treated as test file and we don't even need to use `.test` or `.spec` extension for files under `__tests__` folder
