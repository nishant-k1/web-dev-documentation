## âœ… Step 4: Inside `headerSlice.js` (your component slice)

```js
import { createSlice } from "@reduxjs/toolkit";

// Step 1: Define initial state
const initialState = {
  count: 0,
};

// Step 2: Create the slice
const headerSlice = createSlice({
  name: "header",
  initialState,
  reducers: {
    increment: (state) => {
      state.count += 1;
    },
    decrement: (state) => {
      state.count -= 1;
    },
    setCount: (state, action) => {
      state.count = action.payload;
    },
  },
});

// Step 3: Export action creators
export const { increment, decrement, setCount } = headerSlice.actions;

// Step 4: Export the reducer
export default headerSlice.reducer;
```

âœ… `createSlice` auto-generates:

- Action types (`header/increment`)
- Action creators (`increment()`)
- Reducer logic (with immer)

## âœ… Step 5: Use in Components

```js
import { useSelector, useDispatch } from "react-redux";
import { increment, setCount } from "../state/headerSlice";

function HeaderComponent() {
  // Step 1: Read state using useSelector
  const count = useSelector((state) => state.header.count);

  // Step 2: Get dispatch function
  const dispatch = useDispatch();

  // Step 3: Use dispatch to fire actions
  const handleClick = () => {
    dispatch(increment()); // Action creator returns action object internally
  };

  const handleSet = () => {
    dispatch(setCount(10)); // Pass payload directly
  };

  return (
    <div>
      <h1>{count}</h1>
      <button onClick={handleClick}>Increment</button>
      <button onClick={handleSet}>Set to 10</button>
    </div>
  );
}
```

âœ… Clean. No need to manually write action types, switch cases, etc.

## âœ… Summary: Redux Toolkit vs Classic Redux

| Classic Redux                  | Redux Toolkit (RTK)             |
| ------------------------------ | ------------------------------- |
| `createStore`                  | `configureStore`                |
| `combineReducers`              | Handled inside `configureStore` |

| Manual reducer switch-case     | Auto-created via `createSlice`  |
| Action types & creators manual | Auto-generated by `createSlice` |
| Verbose                        | Minimal boilerplate             |
