# HTML Element Interfaces & Their Properties/Methods

---

## 1. `HTMLElement` (Base for all HTML elements)

### Common to all HTML elements

**Properties:**

- `id`
- `className`
- `style`
- `tabIndex`
- `title`
- `hidden`
- `contentEditable`
- `draggable`
- `accessKey`
- `lang`
- `dir`
- `translate`

**Methods:**

- `focus()`
- `blur()`
- `click()`

**Event Handlers:**  
Event handlers like `onclick`, `oninput`, etc. are technically properties — just special ones.

- `onclick`
- `onfocus`
- `onblur`
- `onmouseenter`
- `onkeydown`
