# JavaScript

# Suggested Learning Order (ECMAScript focus)

1. Expressions vs Statements (`02. Expressions and Statements`)
2. Declarations → Initialization → Assignment (`03. Declarations, Initializations, and Assignments`)
3. Data Types + Type Conversion/Coercion (`05. Data Types`)
4. Operators + Precedence (`06. Operators`)
5. Functions + this-binding + Scope/Hoisting/Closures/Execution Context  
   (combine `07. Functions: this Binding and Function Invocation` + `04. Accessing Declarations` + `08. Execution Context, Hoisting, Scope and Closures`)
6. Strict Mode (`11. Strict Mode`)
7. Programming Paradigms (OOP, FP, comparison/decision) (`09. Programming Paradigm`)
8. Async Language Features (Callbacks → Promises → async/await) (`12. Handling Async Js - Belongs to ECMAScript`)
9. Error Handling & Debugging (sync + async) (`10. Error Handling & Debugging`)
10. Modules, Loading, Bundling/Build Tooling (`13. Modules & Module Loaders - Belongs to ECMAScript | Module Bundlers & Build - Tooling`)
11. Internationalization / `Intl` (`14. INTERNATIONALIZATION - Belongs to ECMAScript (ES)`)
12. ECMAScript Version Reference (ES6–ES15 feature lists) (`15. ECMAScript Versions and Corresponding Features`)

# Core JavaScript / ECMAScript

Core JS = ECMAScript (the language: syntax, data types, functions, promises, classes, modules, etc.).

# Advanced JavaScript (language + runtime)

Advanced JS = Advanced ECMAScript features + JavaScript Runtime Environment (JRE) features (DOM/Browser/Node APIs, event loop, performance, etc.).
