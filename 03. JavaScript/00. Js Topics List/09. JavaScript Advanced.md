# Advanced JavaScript Concepts

## 1. Execution Context & Scope

- **Execution Context Creation Phases**: Creation and Execution phases in JavaScript.
- **Variable Environment vs Lexical Environment**: Differences and how they affect variable resolution.
- **`this` Binding in Different Contexts**: Global scope, function scope, arrow functions, and class methods.

## 2. Closures

- **Definition**: Functions that retain access to their lexical scope.
- **Usage**: Private variables and functions.

## 3. Prototypes and Inheritance

- **Prototypes**: `Object.prototype`
- **Prototype Chain**: `obj.__proto__`
- **Inheritance**: `Object.create()`, `class` syntax

## 4. Higher-Order Functions

- **Definition**: Functions that take other functions as arguments or return functions.
- **Examples**: `map()`, `filter()`, `reduce()`

## 5. Functional Programming

- **Pure Functions**: Functions with no side effects.
- **Immutability**: Data that cannot be changed once created.
- **Function Composition**: Combining functions to create new functions.

## 6. Asynchronous Patterns

- **Callback Functions**: Functions passed as arguments to other functions.
- **Promises**: `new Promise()`, `.then()`, `.catch()`, `.finally()`
- **Async/Await**: Syntactic sugar over promises for more readable async code.
- **Generators**: `function*`, `yield`, `next()`
- **Timers and Asynchronous Behavior**: Differences between `setTimeout`, `setInterval`, and `requestAnimationFrame`
- **Microtasks vs Macrotasks**: Event loop behavior with Promises, `setTimeout`, and asynchronous APIs

## 7. Modules

- **ES6 Modules**: `import`, `export`, `export default`
- **Dynamic Imports**: `import('module')`
- **Module Bundlers**: Webpack, Rollup

## 8. Memory Management

- **Garbage Collection**: Automatic memory management.
- **Memory Leaks**: Unintentional retention of memory.
- **Memory Snapshots**: Analyzing heap allocation and preventing memory leaks

## 9. Design Patterns

- **Singleton**: A class with a single instance.
- **Factory**: A method for creating objects.
- **Observer**: A pattern where objects subscribe to events.
- **Module**: Encapsulation of code.

## 10. Advanced Object Concepts

- **Object Cloning**: Methods for shallow copy (`Object.assign()`, spread operator)
- **Shallow vs Deep Copy**: Differences and when to use deep cloning techniques
- **Sealing and Freezing Objects**: Preventing extensions and mutations using `Object.seal()` and `Object.freeze()`

## 11. Meta-Programming

- **Reflect API**: `Reflect.get()`, `Reflect.set()`
- **Proxy API**: `new Proxy(target, handler)`

## 12. Event Handling

- **Event Capturing, Bubbling, and Delegation**: Understanding the event propagation model
- **`addEventListener` Options**: Usage of `{capture}`, `{once}`, and other options
- **Custom Events**: Creating and dispatching events with `new Event()` or `CustomEvent()`

## 13. Web APIs and Modern Features

- **Fetch API**: `fetch()`, `Response`, `Request`, `Headers`
- **AJAX**: `XMLHttpRequest()`
- **Web Workers**: Background threads for parallel processing
- **Service Workers**: For offline capabilities and caching
- **WebSockets**: Real-time bi-directional communication via `WebSocket` API
- **Web Storage**: `localStorage`, `sessionStorage`, `IndexedDB`
- **Geolocation API**: `navigator.geolocation` for retrieving the user's geographic location
- **Browser-Specific APIs**: Clipboard API, Notification API, Fullscreen API
- **Shadow DOM**: Encapsulation of DOM and styles in Web Components
- **Audio and Video APIs**: Web Audio API, Media Devices API

## 14. Error Handling and Debugging

- **Synchronous vs Asynchronous Error Handling**: Handling errors in both synchronous and async code
- **Using `console` Effectively**: Tools like `console.time`, `console.table`, and other debugging methods
- **Debugging with Browser Developer Tools**: Breakpoints, watch expressions, and profiling

## 15. Strict Mode

- **Enabling Strict Mode**: Using `"use strict"` and its benefits
- **Common Pitfalls Resolved by Strict Mode**: Silent errors, accidental globals, and restricted keywords

## 16. Internationalization

- **Using the `Intl` API**: Formatting dates, numbers, currencies, and strings for various locales

## 17. Polyfills and Transpilation

- **Writing Polyfills**: Implementing browser-compatible versions of modern JavaScript features
- **Role of Transpilers**: Tools like Babel for converting modern JS to compatible versions

## 18. Performance Optimization

- **Debouncing/Throttling**: Reducing the frequency of function calls
- **Code Splitting**: Loading parts of the application on demand
- **Lazy Loading**: Loading resources only when needed
- **Profiling with Browser Developer Tools**: Monitoring function execution times and optimizing code

## 19. Patterns and Architecture

- **Event Emitters**: Pattern for subscribing and listening to events, especially in Node.js
- **Module Patterns**: ES5 Revealing Module Pattern and ES6 Module Syntax

## 20. Browser Internals

- **Event Loop**: How JavaScript handles asynchronous code
- **Call Stack**: Execution context and stack frames
- **Rendering Pipeline**: How browsers render pages

## 21. TypeScript and Type Checking

- **Type Annotations**: Static type checking with TypeScript
- **Interfaces**: Defining contracts for objects
- **Generics**: Type-safe functions and classes

## 22. Security

- **Cross-Site Scripting (XSS)**
- **Cross-Site Request Forgery (CSRF)**
- **Content Security Policy (CSP)**
