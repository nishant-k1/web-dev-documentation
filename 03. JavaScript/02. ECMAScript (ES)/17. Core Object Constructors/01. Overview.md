# Built-in Constructors

These are **built-in constructors**: they are functions you can use with `new` (e.g. `new Array()`, `new Object()`) or via literals (`[]`, `{}`, `""`). In JavaScript, every function is an object, so they also have static methods on the constructor (e.g. `Object.keys`, `Array.isArray`).

See parent folder [10. Standard Built-in Objects](../10.%20Standard%20Built-in%20Objects/) for the full overview and study order.
