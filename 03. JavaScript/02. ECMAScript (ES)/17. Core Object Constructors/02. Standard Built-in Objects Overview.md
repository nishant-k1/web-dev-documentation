# ✅ Standard Built-in JavaScript Objects

**Built-in constructors (and objects):** `Array`, `Object`, `String`, `Number`, `Boolean`, `Date`, `RegExp`, `Error`, `Set`, `Map`, `Promise`, etc. are **built-in constructors**. In JavaScript, every function is an object, so these constructors are also **objects** with their own properties and methods (e.g. `Object.keys`, `Array.isArray`). You use them with `new` or via literals (`[]`, `{}`, `""`), and you call static methods on the constructor itself.

**Built-in objects (not constructors):** `Math`, `JSON`, `Reflect`, `Symbol`, `BigInt`, `Intl` are **built-in objects** that are **not** constructors — you never use `new Math()` or `new JSON()`. They only provide static properties and methods (e.g. `Math.PI`, `JSON.stringify()`). They do not have a `.prototype` used for instances.
