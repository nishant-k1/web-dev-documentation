# Spread Syntax (`...`)

**Purpose:** Expands (spreads out) iterable values into individual elements.
**Main Application**: Immutable update = copy → modify → return new value (without touching original)

## Spread in React (Common Use Case)

1. Spreading props

   ```javascript
   function Button(props) {
     return <button {...props}>Click me</button>;
   }
   ```

2. Merging props

   ```javascript
   function CustomButton({ className, ...otherProps }) {
     return <button className={`btn ${className}`} {...otherProps} />;
   }
   ```
