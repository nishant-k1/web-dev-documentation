# Built-in Iterables

JavaScript has many built-in iterables that work with `for...of`, spread operator, and destructuring.

---

## Arrays

Arrays are iterable by default:

```javascript
const arr = [1, 2, 3];

// for...of
for (const value of arr) {
  console.log(value); // 1, 2, 3
}

// Spread
const copy = [...arr]; // [1, 2, 3]

// Destructuring
const [first, ...rest] = arr;
```

---

## Strings

Strings are iterable (iterates over characters):

```javascript
const str = "hello";

for (const char of str) {
  console.log(char); // h, e, l, l, o
}

const chars = [...str]; // ['h', 'e', 'l', 'l', 'o']
```

---

## Maps

Maps are iterable (iterates over [key, value] pairs):

```javascript
const map = new Map([
  ["a", 1],
  ["b", 2],
]);

for (const [key, value] of map) {
  console.log(key, value); // a 1, b 2
}
```

---

## Sets

Sets are iterable (iterates over values):

```javascript
const set = new Set([1, 2, 3]);

for (const value of set) {
  console.log(value); // 1, 2, 3
}
```

---

## NodeLists

DOM NodeLists are iterable:

```javascript
const nodes = document.querySelectorAll("div");

for (const node of nodes) {
  console.log(node);
}
```

---

## Arguments Object

The `arguments` object is iterable:

```javascript
function test() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

test(1, 2, 3); // 1, 2, 3
```

---

## TypedArrays

TypedArrays (Uint8Array, Int16Array, etc.) are iterable:

```javascript
const typedArray = new Uint8Array([1, 2, 3]);

for (const value of typedArray) {
  console.log(value); // 1, 2, 3
}
```

---

## What is NOT Iterable?

- **Plain Objects**: `{}` is not iterable by default
- **Numbers, Booleans, null, undefined**: Not iterable

---

## Making Objects Iterable

To make a plain object iterable, add `Symbol.iterator`:

```javascript
const obj = {
  a: 1,
  b: 2,
  [Symbol.iterator]() {
    const keys = Object.keys(this);
    let index = 0;
    return {
      next: () => {
        if (index < keys.length) {
          return {
            value: this[keys[index++]],
            done: false,
          };
        }
        return { done: true };
      },
    };
  },
};

for (const value of obj) {
  console.log(value); // 1, 2
}
```
