# Block Statements

Group multiple statements together using curly braces `{}`.

## Syntax

```js
{
  statement1;
  statement2;
  statement3;
}
```

## Examples

```js
// Simple block
{
  let x = 10;
  let y = 20;
  console.log(x + y);
}

// Block in if statement
if (condition) {
  let x = 10;
  let y = 20;
  console.log(x + y);
}

// Block in function
function example() {
  let x = 10;
  let y = 20;
  return x + y;
}
```

## Scope Creation

Block statements create a new scope for `let`, `const`, and `class` declarations.

```js
{
  let x = 10;
  const y = 20;
  // x and y are only accessible within this block
}

// x and y are not accessible here
// console.log(x); // ReferenceError
```

## Important Notes

- Block statements `{}` create new scope for `let`, `const`, and `class`
- `var` declarations are function-scoped, not block-scoped
- Blocks can be nested
- Empty blocks `{}` are valid statements
- Block statements have completion values (usually `undefined`)

## Scope Examples

```js
// var is function-scoped
function example() {
  if (true) {
    var x = 10; // Accessible throughout the function
  }
  console.log(x); // 10 - works!
}

// let is block-scoped
function example2() {
  if (true) {
    let y = 20; // Only accessible in this block
  }
  // console.log(y); // ReferenceError - y is not accessible
}
```
