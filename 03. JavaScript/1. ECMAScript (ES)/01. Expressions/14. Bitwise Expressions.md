# Bitwise Expressions

Perform bitwise operations.

## Bitwise AND

```js
5 & 3; // 1
```

## Bitwise OR

```js
5 | 3; // 7
```

## Bitwise XOR

```js
5 ^ 3; // 6
```

## Bitwise NOT

```js
~5; // -6
```

## Left Shift

```js
5 << 1; // 10
```

## Right Shift

```js
5 >> 1; // 2
```

## Unsigned Right Shift

```js
-5 >>> 1; // 2147483645
```

## Important Notes

- **Bitwise operations**: Operate on the binary representation of numbers
- **Use cases**: Often used for flags, masks, and low-level optimizations
- **Performance**: Generally faster than arithmetic operations, but less readable
