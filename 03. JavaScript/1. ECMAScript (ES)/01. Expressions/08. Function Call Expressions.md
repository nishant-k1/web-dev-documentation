# Function Call Expressions

Invoke functions and methods.

## Function Call

```js
// Function call
myFunction(); // Calls function
myFunction(1, 2, 3); // Calls with arguments
```

## Method Call

```js
obj.method(); // Calls method on object
array.push(item); // Calls method on array
```

## Constructor Call

```js
new Date(); // Creates new instance
new Array(5); // Creates new array
```

## Important Notes

- **Return value**: Function calls return the value specified by the function's return statement
- **Arguments**: Functions can accept zero or more arguments
- **Method calls**: Methods are functions that are properties of objects
