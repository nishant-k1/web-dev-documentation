# Yield Expression

Used in generator functions to pause execution and yield values.

## Syntax

```js
yield value;
yield* iterable;
```

## Examples

```js
function* generator() {
  yield 1;
  yield 2;
  yield 3;
}

const gen = generator();
gen.next(); // { value: 1, done: false }
gen.next(); // { value: 2, done: false }
gen.next(); // { value: 3, done: false }
gen.next(); // { value: undefined, done: true }
```

## Yield with Value

```js
function* generator() {
  let value = yield 1;
  yield value * 2;
}
```

## Yield*

Delegates to another generator:

```js
function* gen1() {
  yield 1;
  yield 2;
}

function* gen2() {
  yield* gen1();
  yield 3;
}
```

## Important Notes

- **Generator functions**: Can only be used inside generator functions (`function*`)
- **Pause execution**: Pauses function execution and returns a value
- **Resumable**: Function can be resumed from where it paused
- **Iterator protocol**: Generators implement the iterator protocol
