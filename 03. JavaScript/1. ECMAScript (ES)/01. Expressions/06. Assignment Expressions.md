# Assignment Expressions

Assign values to variables.

## Simple Assignment

```js
x = 5; // Assigns 5 to x
```

## Compound Assignment

```js
x += 5; // x = x + 5
x -= 3; // x = x - 3
x *= 2; // x = x * 2
x /= 2; // x = x / 2
x %= 3; // x = x % 3
x **= 2; // x = x ** 2
```

## Bitwise Assignment

```js
x &= 5; // x = x & 5
x |= 3; // x = x | 3
x ^= 2; // x = x ^ 2
x <<= 1; // x = x << 1
x >>= 1; // x = x >> 1
x >>>= 1; // x = x >>> 1
```

## Assignment Chaining

Assignment expressions return the assigned value, allowing chaining:

```js
let a, b, c;
a = b = c = 5; // All set to 5 (right-to-left evaluation)
```

## Important Notes

- **Return value**: Assignment expressions return the assigned value
- **Evaluation order**: Assignment is right-associative (evaluates right-to-left)
- **Side effects**: Assignment expressions modify variables (side effect)
