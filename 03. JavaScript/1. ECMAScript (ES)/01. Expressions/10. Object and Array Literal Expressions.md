# Object and Array Literal Expressions

Create objects and arrays.

## Object Literal

```js
let obj = {
  name: "John",
  age: 30,
  greet() {
    return "Hello";
  },
};
```

## Array Literal

```js
let arr = [1, 2, 3, 4, 5];
```

## Nested Structures

```js
let nested = {
  users: [
    { name: "Alice", age: 25 },
    { name: "Bob", age: 30 },
  ],
};
```

## Important Notes

- **Object literals**: Create objects using curly braces `{}`
- **Array literals**: Create arrays using square brackets `[]`
- **Nested structures**: Objects and arrays can contain other objects and arrays
