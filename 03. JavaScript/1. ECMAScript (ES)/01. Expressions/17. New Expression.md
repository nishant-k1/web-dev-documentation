# New Expression

Creates new instances using constructors.

## Examples

```js
new Date(); // New Date object
new Array(5); // New array with 5 empty slots
new Object(); // New empty object
new MyClass(); // New instance of custom class
```

## Constructor Functions

```js
function Person(name) {
  this.name = name;
}

let person = new Person("John");
```

## Important Notes

- **Instance creation**: Creates a new instance of an object
- **Constructor**: Calls the constructor function with the provided arguments
- **this binding**: In constructors, `this` refers to the newly created instance
