# Logical Expressions

Perform logical operations using `&&`, `||`, and `!`.

## Logical AND (&&)

```js
true && false; // false
5 > 3 && 10 > 5; // true
```

## Logical OR (||)

```js
true || false; // true
5 < 3 || 10 > 5; // true
```

## Logical NOT (!)

```js
!true; // false
!false; // true
```

## Short-Circuit Evaluation

Logical operators use short-circuit evaluation:

```js
// AND short-circuits on first falsy value
null && "hello"; // null (short-circuits, returns first falsy)

// OR short-circuits on first truthy value
"hello" || "world"; // "hello" (short-circuits, returns first truthy)
```

## Important Notes

- **Short-circuit evaluation**: Logical operators may not evaluate all operands
- **Return values**: `&&` and `||` return the last evaluated operand, not necessarily a boolean
- **Common pattern**: Used for default values and conditional execution
