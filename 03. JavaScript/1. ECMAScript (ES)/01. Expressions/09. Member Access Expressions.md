# Member Access Expressions

Access properties and methods of objects.

## Dot Notation

```js
obj.property; // Access property
obj.method(); // Access method
```

## Bracket Notation

```js
obj["property"]; // Access property (dynamic)
obj[computedKey]; // Access with computed key
array[0]; // Access array element
```

## Optional Chaining (ES2020)

```js
obj?.property; // Safe property access
obj?.method?.(); // Safe method call
```

## Important Notes

- **Dot notation**: Used when property name is known and valid identifier
- **Bracket notation**: Used for dynamic property names or when property name is not a valid identifier
- **Optional chaining**: Returns `undefined` if any part of the chain is `null` or `undefined`, preventing errors
