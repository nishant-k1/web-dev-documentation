# Primary Expressions

The simplest forms of expressions - literals, identifiers, and keywords.

## Literals

```js
42;                    // Number literal
"hello";               // String literal
true;                  // Boolean literal
null;                  // Null literal
undefined;             // Undefined literal
[1, 2, 3];            // Array literal
{ x: 1, y: 2 };       // Object literal
/pattern/;            // Regular expression literal
```

## Identifiers

```js
x; // Variable reference
myFunction; // Function name reference
Math; // Object reference
```

## Keywords

```js
this;                 // Refers to current object context
super;                // Refers to parent class (in classes)
```
