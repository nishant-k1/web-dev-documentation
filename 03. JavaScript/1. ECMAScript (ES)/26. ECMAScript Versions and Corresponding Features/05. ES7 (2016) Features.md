## Major ES7 Features

### **Async/Await**

ğŸ“– See: [21. Handling Async Js](../21.%20Handling%20Async%20Js%20-%20Belongs%20to%20ECMAScript/01.%20Handling%20Async%20Js.md)

Syntactic sugar for working with Promises, making asynchronous code look synchronous.

```javascript
async function fetchData() {
  const data = await fetch("/api/data");
  return data.json();
}
```

### **Object.entries, Object.values**

ğŸ“– See: [04. Data Types - Objects](../04.%20Data%20Types/12.%20Objects.md)

Methods to iterate over object properties.

```javascript
Object.entries({ a: 1, b: 2 }); // [["a", 1], ["b", 2]]
Object.values({ a: 1, b: 2 }); // [1, 2]
```
