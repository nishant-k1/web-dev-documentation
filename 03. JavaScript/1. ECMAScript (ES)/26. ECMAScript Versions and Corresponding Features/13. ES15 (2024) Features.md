# ES15 (ECMAScript 2024) features are still under development and subject to change. However, some proposed features

- **Array.prototype.findLast and Array.prototype.findLastIndex: Find the last element or index of an element in an array that satisfies a test function.**
- **String.prototype.fromCodePoint: Creates a new string from a sequence of Unicode code points.**
- **Object.isFrozen: Checks if an object is frozen.**
- **Reflect.isFrozen: Checks if an object is frozen, similar to Object.isFrozen.**
- **RegExp.prototype.testAll: Tests a regular expression against a string and returns all matches.**
- **Intl.DateTimeFormat: Improved formatting for dates and times.**
- **WebAssembly SIMD: Additional features and improvements to WebAssembly SIMD.**
- **Temporal API: Further enhancements to the Temporal API for working with dates and times.**
