## ðŸ”¹ Math Rounding Methods

| Method          | Description                             |
| --------------- | --------------------------------------- |
| `Math.round(x)` | Rounds to nearest integer               |
| `Math.ceil(x)`  | Rounds up to the next integer           |
| `Math.floor(x)` | Rounds down to the previous integer     |
| `Math.trunc(x)` | Removes fractional digits (toward zero) |

```js
Math.round(4.6); // 5
Math.floor(4.9); // 4
Math.trunc(-4.9); // -4
```

## ðŸ§  Interview Patterns Using `Math`

- Generate random integer in a range:

  ```js

    return Math.floor(Math.random() * (max - min + 1)) + min;
  }
  ```

- Clamp a value between a min and max:

  ```js
  Math.max(min, Math.min(value, max));
  ```

- Round to N decimal places:
  ```js

    return Number(num.toFixed(decimals));
  }
  ```
