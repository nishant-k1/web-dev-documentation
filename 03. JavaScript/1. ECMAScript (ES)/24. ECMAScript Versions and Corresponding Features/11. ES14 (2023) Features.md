# ES14 (ECMAScript 2023) features

- **Pattern Matching: Introduces a new pattern matching syntax for destructuring assignments and conditional expressions.**
- **Array.prototype.isTemplateObject: Checks if an object is a template object.**
- **String.prototype.at: Retrieves an element from a string by its index, supporting negative indices.**
- **Object.hasOwn: Determines if an object has its own property.**
- **RegExp Match Indices: Provides more information about the indices of matched groups in regular expressions.**
- **Intl.NumberFormat: Improved formatting for currency and other numbers.**
- **WebAssembly SIMD: Additional features and improvements to WebAssembly SIMD.**
- **Temporal API: Further enhancements to the Temporal API for working with dates and times.**
