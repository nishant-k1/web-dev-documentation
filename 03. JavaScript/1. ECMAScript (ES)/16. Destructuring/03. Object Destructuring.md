# Object Destructuring

Object destructuring allows you to extract properties from objects based on their **key names**.

---

## Basic Syntax

```javascript
const user = { name: "John", age: 30, city: "NYC" };
const { name, age } = user;

console.log(name); // "John"
console.log(age); // 30
```

---

## Key Characteristics

- **Key names matter**: Properties are extracted by key name, not position
- **Order doesn't matter**: Variables can be in any order
- **Can rename**: Assign to different variable names
- **Can use rest**: Capture remaining properties

---

## Common Patterns

### 1. Basic Extraction

```javascript
const { name, age } = { name: "John", age: 30 };
```

### 2. Renaming Variables

```javascript
const { name: userName, age: userAge } = { name: "John", age: 30 };
console.log(userName); // "John"
console.log(userAge); // 30
```

### 3. Default Values

```javascript
const { name, age = 25 } = { name: "John" };
console.log(age); // 25 (default)
```

### 4. Renaming + Defaults

```javascript
const { name: userName = "Guest", age: userAge = 0 } = { name: "John" };
console.log(userName); // "John"
console.log(userAge); // 0 (default)
```

### 5. Rest Operator

```javascript
const { name, ...rest } = { name: "John", age: 30, city: "NYC" };
console.log(name); // "John"
console.log(rest); // { age: 30, city: "NYC" }
```

### 6. Nested Objects

```javascript
const {
  user: {
    name,
    profile: { age },
  },
} = {
  user: { name: "John", profile: { age: 30 } },
};
console.log(name); // "John"
console.log(age); // 30
```

### 7. Computed Property Names

```javascript
const key = "name";
const { [key]: value } = { name: "John" };
console.log(value); // "John"
```

---

## Use Cases

| Use Case                       | Example                                  |
| ------------------------------ | ---------------------------------------- |
| Extracting specific properties | `const { name, email } = user;`          |
| API response handling          | `const { data, status } = response;`     |
| Configuration objects          | `const { host, port } = config;`         |
| Function parameters            | `function greet({ name }) {}`            |
| Avoiding repetitive code       | Instead of `user.name`, `user.age`, etc. |
