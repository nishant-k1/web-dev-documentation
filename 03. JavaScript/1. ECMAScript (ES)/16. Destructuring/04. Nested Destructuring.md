# Nested Destructuring

Nested destructuring allows you to extract values from nested arrays and objects.

---

## Nested Array Destructuring

```javascript
const arr = [1, [2, 3], 4];
const [a, [b, c], d] = arr;

console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
console.log(d); // 4
```

---

## Nested Object Destructuring

```javascript
const user = {
  name: "John",
  address: {
    city: "NYC",
    country: "USA",
  },
};

const {
  name,
  address: { city, country },
} = user;

console.log(name); // "John"
console.log(city); // "NYC"
console.log(country); // "USA"
// Note: 'address' is not available (it was destructured)
```

---

## Mixed Nested Destructuring

```javascript
const data = {
  users: [
    { name: "John", age: 30 },
    { name: "Jane", age: 25 },
  ],
};

const {
  users: [first, second],
} = data;
const {
  users: [{ name: firstName }, { name: secondName }],
} = data;

console.log(firstName); // "John"
console.log(secondName); // "Jane"
```

---

## Default Values in Nested Destructuring

```javascript
const user = {
  name: "John",
  // address is missing
};

const { name, address: { city = "Unknown" } = {} } = user;

// Error! Can't destructure from undefined
// Need to provide default for address first:

const { name, address: { city = "Unknown" } = {} } = user;
```
