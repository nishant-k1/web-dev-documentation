# Built-in Objects (Not Constructors)

- These are built-in objects that are not constructors: you never use `new` with them (E.g. no `new Math()`, no `new JSON()`) to create instance.
- They only provide static properties and methods. E.g. `Math.PI`, `Math.random()`, `JSON.stringify()`, `JSON.parse()`
- They do not have a `.prototype` used for instances.
