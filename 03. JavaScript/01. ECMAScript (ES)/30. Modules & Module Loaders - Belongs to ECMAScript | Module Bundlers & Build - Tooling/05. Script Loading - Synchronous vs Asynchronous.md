## How HTML Elements Load: Synchronous vs Asynchronous

Every browser follows a strict loading timeline when parsing HTML.
We will divide elements into 3 categories:

    1. Synchronous / Blocking
    2. Asynchronous / Non-blocking
    3. Hybrid (sometimes blocking, sometimes async)

**1. Synchronous (Blocking) Elements**
These elements stop HTML parsing until they finish loading or processing.
