# ‚úÖ Must-Know `ArrayBuffer` Object

`ArrayBuffer` represents raw binary data of fixed length. It cannot be directly read or written - you need a view (TypedArray or DataView).

---

## Creating ArrayBuffer

```js
// Create buffer with specified size (in bytes)
const buffer = new ArrayBuffer(16); // 16 bytes
console.log(buffer.byteLength); // 16

// Buffer is initialized with zeros
```

---

## ArrayBuffer Properties

```js
const buffer = new ArrayBuffer(16);

buffer.byteLength; // 16 (read-only)
buffer.slice(start, end); // Creates new buffer (copies bytes)
```

---

## Slicing ArrayBuffer

```js
const buffer = new ArrayBuffer(16);
const slice1 = buffer.slice(0, 8); // First 8 bytes
const slice2 = buffer.slice(8); // Bytes 8 to end

// Slices are copies, not views
```

---

## Checking if ArrayBuffer

```js
const buffer = new ArrayBuffer(16);

buffer instanceof ArrayBuffer; // true
buffer.constructor === ArrayBuffer; // true
```

---

## Common Use Cases

### 1. Creating Typed Array Views

```js
const buffer = new ArrayBuffer(16);

// Different views of same buffer
const int8View = new Int8Array(buffer); // 16 elements
const int16View = new Int16Array(buffer); // 8 elements
const int32View = new Int32Array(buffer); // 4 elements
const float32View = new Float32Array(buffer); // 4 elements
```

### 2. Binary Data from Files

```js
async function readBinaryFile(file) {
  const buffer = await file.arrayBuffer();
  // Process buffer
  const uint8 = new Uint8Array(buffer);
}
```

### 3. Network Data

```js
async function fetchBinaryData(url) {
  const response = await fetch(url);
  const buffer = await response.arrayBuffer();
  return buffer;
}
```

### 4. Creating Copies

```js
const buffer1 = new ArrayBuffer(16);
const buffer2 = buffer1.slice(); // Full copy
const buffer3 = buffer1.slice(0, 8); // Partial copy
```

## Important Notes

### ‚ö†Ô∏è Cannot Directly Access

ArrayBuffer cannot be read or written directly. You must use:

- Typed Arrays (Int8Array, Uint8Array, Float32Array, etc.)
- DataView

### ‚úÖ Memory Efficiency

ArrayBuffer provides efficient, contiguous memory allocation for binary data.

## Interview Common Questions

### Q: What is ArrayBuffer?

Raw binary data container of fixed length. Cannot be directly accessed - requires a view (TypedArray or DataView) to read/write.

### Q: How to create a copy of ArrayBuffer?

```js
const copy = buffer.slice();
```

### Q: How to get size of ArrayBuffer?

```js
buffer.byteLength; // Size in bytes
```
