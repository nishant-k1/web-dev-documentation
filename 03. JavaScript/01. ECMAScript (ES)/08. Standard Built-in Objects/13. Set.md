# âœ… Must-Know `Set` Object

`Set` is a collection of **unique values** (no duplicates allowed). ES6 feature.

---

## Creating Set

```js
// Empty Set
const set = new Set();

// From array (removes duplicates automatically)
const set = new Set([1, 2, 3, 2, 1]); // Set(3) {1, 2, 3}

// From string
const set = new Set("hello"); // Set(4) {'h', 'e', 'l', 'o'}

// From iterable
const set = new Set([1, 2, 3].map((x) => x * 2)); // Set(3) {2, 4, 6}
```

---

## Set Methods

| Method              | Description            | Returns           |
| ------------------- | ---------------------- | ----------------- |
| `set.add(value)`    | Adds value to set      | `Set` (chainable) |
| `set.has(value)`    | Checks if value exists | `boolean`         |
| `set.delete(value)` | Removes value          | `boolean`         |
| `set.clear()`       | Removes all values     | `undefined`       |
| `set.size`          | Number of values       | `number`          |

## ES2024 Set Methods â­

### `union(other)` - Combine sets

```js
const set1 = new Set([1, 2, 3]);
const set2 = new Set([3, 4, 5]);
const union = set1.union(set2); // Set(5) {1, 2, 3, 4, 5}
```

### `intersection(other)` - Common values

```js
const set1 = new Set([1, 2, 3]);
const set2 = new Set([2, 3, 4]);
const intersection = set1.intersection(set2); // Set(2) {2, 3}
```

### `difference(other)` - Values in this but not other

```js
const set1 = new Set([1, 2, 3]);
const set2 = new Set([2, 3, 4]);
const difference = set1.difference(set2); // Set(1) {1}
```

### `symmetricDifference(other)` - Values in either but not both

```js
const set1 = new Set([1, 2, 3]);
const set2 = new Set([2, 3, 4]);
const symDiff = set1.symmetricDifference(set2); // Set(2) {1, 4}
```

### `isSubsetOf(other)` - Is this a subset?

```js
const set1 = new Set([1, 2]);
const set2 = new Set([1, 2, 3]);
set1.isSubsetOf(set2); // true
```

### `isSupersetOf(other)` - Is this a superset?

```js
const set1 = new Set([1, 2, 3]);
const set2 = new Set([1, 2]);
set1.isSupersetOf(set2); // true
```

### `isDisjointFrom(other)` - No common values?

```js
const set1 = new Set([1, 2]);
const set2 = new Set([3, 4]);
set1.isDisjointFrom(set2); // true
```

## Iteration

```js
const set = new Set([1, 2, 3]);

// for...of
for (const value of set) {
  console.log(value);
}

// forEach
set.forEach((value) => {
  console.log(value);
});

// Iterator methods (all return same iterator)
set.keys(); // SetIterator {1, 2, 3}
set.values(); // SetIterator {1, 2, 3}
set.entries(); // SetIterator {[1, 1], [2, 2], [3, 3]}

// Spread operator
[...set]; // [1, 2, 3]
```

## Common Use Cases

### 1. Remove Duplicates

```js
const arr = [1, 2, 2, 3, 3, 3];
const unique = [...new Set(arr)]; // [1, 2, 3]
```

### 2. Track Unique Items

```js
const visited = new Set();
function visit(url) {
  if (visited.has(url)) return;
  visited.add(url);
  // Process URL
}
```

### 3. Set Operations (ES2024)

```js
// Find common elements
const users1 = new Set(["alice", "bob", "charlie"]);
const users2 = new Set(["bob", "david", "charlie"]);
const common = users1.intersection(users2); // {'bob', 'charlie'}
```

### 4. Membership Checking

```js
const allowedDomains = new Set(["example.com", "test.com"]);
if (allowedDomains.has(domain)) {
  // Allow access
}
```

## Set vs Array

| Feature         | Set                 | Array             |
| --------------- | ------------------- | ----------------- |
| **Uniqueness**  | Automatic           | Manual check      |
| **Index**       | No indexing         | Indexed access    |
| **Order**       | Insertion order     | Index order       |
| **Performance** | O(1) has/add/delete | O(n) for includes |
| **Use Case**    | Unique collections  | Ordered lists     |

## Interview Common Questions

### Q: How to remove duplicates from array?

```js
const unique = [...new Set(array)];
```

### Q: Difference between Set and Array?

Set automatically enforces uniqueness and has O(1) lookup/insert/delete. Array allows duplicates and has indexed access.

### Q: Can Set have objects as values?

Yes! But object equality is by reference, not value.

```js
const set = new Set();
set.add({ id: 1 });
set.add({ id: 1 }); // Different object, both added
set.size; // 2
```
