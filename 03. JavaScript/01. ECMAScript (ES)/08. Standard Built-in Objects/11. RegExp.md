# ‚úÖ Must-Know `RegExp` Object

`RegExp` (Regular Expression) is a built-in object for pattern matching in strings.

---

## Creating RegExp

### Literal Notation (Recommended)

```js
const regex = /pattern/flags;
const regex = /hello/; // No flags
const regex = /hello/gi; // Global and case-insensitive
```

### Constructor Notation

```js
const regex = new RegExp("pattern", "flags");
const regex = new RegExp("hello", "gi");
```

### Dynamic Patterns

```js
// Constructor allows dynamic patterns
const pattern = "hello";
const regex = new RegExp(pattern, "gi");
```

## RegExp Methods

### `test(string)` ‚≠ê‚≠ê‚≠ê

Returns `true` if pattern matches, `false` otherwise.

```js
const regex = /hello/;
regex.test("hello world"); // true
regex.test("world"); // false
```

### `exec(string)` ‚≠ê‚≠ê

Returns match array or `null`. More detailed than `test()`.

```js
const regex = /(\w+)\s(\w+)/;
const result = regex.exec("hello world");
// ['hello world', 'hello', 'world', index: 0, input: 'hello world', groups: undefined]
```

## String Methods That Use RegExp

### `match(regex)` ‚≠ê‚≠ê‚≠ê

```js
"hello world".match(/hello/); // ['hello']
"hello hello".match(/hello/g); // ['hello', 'hello'] (with g flag)
```

### `matchAll(regex)` ‚≠ê‚≠ê (ES2020)

Returns iterator of all matches with groups.

```js
const regex = /t(e)(st(\d?))/g;
const str = "test1test2";
const matches = str.matchAll(regex);
for (const match of matches) {
  console.log(match[0]); // 'test1', 'test2'
  console.log(match[1]); // 'e', 'e'
}
```

### `replace(regex, replacement)` ‚≠ê‚≠ê‚≠ê

```js
"hello world".replace(/world/, "universe"); // "hello universe"
"hello hello".replace(/hello/g, "hi"); // "hi hi"
```

### `search(regex)` ‚≠ê

Returns index of first match or -1.

```js
"hello world".search(/world/); // 6
"hello world".search(/xyz/); // -1
```

### `split(regex)` ‚≠ê

```js
"a,b,c".split(/,/); // ['a', 'b', 'c']
"hello world".split(/\s+/); // ['hello', 'world']
```

## RegExp Flags

| Flag | Name             | Description                               |
| ---- | ---------------- | ----------------------------------------- |
| `g`  | Global           | Find all matches, not just first          |
| `i`  | Case-insensitive | Case-insensitive matching                 |
| `m`  | Multiline        | `^` and `$` match line breaks             |
| `s`  | Dotall           | `.` matches newlines                      |
| `u`  | Unicode          | Unicode-aware matching                    |
| `y`  | Sticky           | Matches only at lastIndex                 |
| `d`  | HasIndices       | Adds indices property to results (ES2022) |

## Common Patterns

### Character Classes

```js
/\d/; // Digit [0-9]
/\w/; // Word character [a-zA-Z0-9_]
/\s/; // Whitespace
/./; // Any character (except newline)
/[abc]/; // Any of a, b, or c
/[^abc]/; // Not a, b, or c
/[a-z]/; // Range a to z
```

### Quantifiers

```js
/a*/; // 0 or more
/a+/; // 1 or more
/a?/; // 0 or 1
/a{3}/; // Exactly 3
/a{3,}/; // 3 or more
/a{3,5}/; // 3 to 5
```

### Anchors

```js
/^start/; // Start of string
/end$/; // End of string
/\bword\b/; // Word boundary
```

### Groups

```js
/(\d+)/; // Capturing group
/(?:\d+)/; // Non-capturing group
/(?<name>\d+)/; // Named group (ES2018)
```

## RegExp Properties

```js
const regex = /hello/g;

regex.source; // "hello" (pattern)
regex.flags; // "g" (flags)
regex.global; // true
regex.ignoreCase; // false
regex.multiline; // false
regex.lastIndex; // 0 (for global/sticky)
```

## Common Use Cases

### 1. Validation

```js
// Email validation (simplified)
const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
emailRegex.test("user@example.com"); // true

// Phone number
const phoneRegex = /^\d{10}$/;
phoneRegex.test("1234567890"); // true
```

### 2. Extraction

```js
const text = "Price: $99.99";
const priceRegex = /\$(\d+\.\d+)/;
const match = text.match(priceRegex);
const price = match[1]; // "99.99"
```

### 3. Replacement

```js
// Replace all digits
"abc123def456".replace(/\d/g, ""); // "abcdef"

// Format phone number
"1234567890".replace(/(\d{3})(\d{3})(\d{4})/, "($1) $2-$3");
// "(123) 456-7890"
```

### 4. Splitting

```js
"one,two,three".split(/,/); // ['one', 'two', 'three']
"a  b   c".split(/\s+/); // ['a', 'b', 'c']
```

## Interview Common Questions

### Q: Difference between `test()` and `exec()`?

- `test()`: Returns boolean (true/false)
- `exec()`: Returns match array with details or null

### Q: What does the `g` flag do?

Makes regex global - finds all matches, not just the first.

### Q: How to match email addresses?

```js
const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
```

(Simplified - real validation is more complex)

### Q: How to extract numbers from string?

```js
"Price: $99.99".match(/\d+\.\d+/); // ['99.99']
```
