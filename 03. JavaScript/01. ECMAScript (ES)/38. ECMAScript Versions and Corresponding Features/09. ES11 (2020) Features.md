## Major ES11 Features

### **Promise.allSettled**

ðŸ“– See: [22. Handling Async Js - Promises](../21.%20Handling%20Async%20Js%20-%20Belongs%20to%20ECMAScript/03.%20Promises.md)

Waits for all promises to settle (fulfilled or rejected) and returns results for all.

```javascript
Promise.allSettled([promise1, promise2]).then((results) => {
  results.forEach((result) => {
    if (result.status === "fulfilled") {
      console.log("Success:", result.value);
    } else {
      console.log("Failed:", result.reason);
    }
  });
});
```

### **GlobalThis**

ðŸ“– See: [04. Data Types - Objects](../04.%20Data%20Types/)

Provides a standard way to access the global object across all environments.

```javascript
// Works in browser, Node.js, Web Workers, etc.
globalThis.setTimeout(() => {}, 1000);
```

### **Numeric Separator**

ðŸ“– See: [04. Data Types - Primitives](../04.%20Data%20Types/01.%20Primitives.md)

Underscores can be used to separate digits in numbers for readability.

```javascript
const million = 1_000_000;
const binary = 0b1010_0001;
```

### **Private Class Fields**

ðŸ“– See: [20. OOP Paradigm - Classes](../19.%20OOP%20Paradigm/08.%20Class%20Pattern.md) | [03. Declarations - Class Declarations](<../03.%20Declarations/04.%20Class%20Declarations%20(class%20keyword).md>)
