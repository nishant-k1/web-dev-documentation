## Major ES13 Features

### **Class Fields (Public and Private)**

ðŸ“– See: [20. OOP Paradigm - Classes](../19.%20OOP%20Paradigm/08.%20Class%20Pattern.md) | [03. Declarations - Class Declarations](<../03.%20Declarations/04.%20Class%20Declarations%20(class%20keyword).md>)

Simplified syntax for defining class fields, including static fields and private fields.

```javascript
class MyClass {
  // Public field
  publicField = 1;

  // Private field
  #privateField = 2;

  // Static field
  static staticField = 3;

  // Private static field
  static #privateStaticField = 4;
}
```

### **Top-Level Await**

ðŸ“– See: [22. Handling Async Js](../21.%20Handling%20Async%20Js%20-%20Belongs%20to%20ECMAScript/) | [24. Modules](../23.%20Modules%20%26%20Module%20Loaders%20-%20Belongs%20to%20ECMAScript%20%7C%20Module%20Bundlers%20%26%20Build%20-%20Tooling/)

Allows `await` at the top level of modules (outside async functions).

```javascript
// In a module
const data = await fetch("/api/data").then((r) => r.json());
export { data };
```

---

### **Array.prototype.at**

ðŸ“– See: [04. Data Types - Arrays](../04.%20Data%20Types/)

Retrieves an element from an array by its index, supporting negative indices.

```javascript
const arr = [1, 2, 3, 4];
arr.at(0); // 1
arr.at(-1); // 4 (last element)
```

---

### **Object.hasOwn**

ðŸ“– See: [04. Data Types - Objects](../04.%20Data%20Types/12.%20Objects.md)

Safer alternative to `Object.prototype.hasOwnProperty`.

```javascript
Object.hasOwn(obj, "property"); // Better than obj.hasOwnProperty('property')
```

### **Class Static Blocks**

ðŸ“– See: [20. OOP Paradigm - Classes](../19.%20OOP%20Paradigm/08.%20Class%20Pattern.md)

Allows initialization logic for static class members.

```javascript
class MyClass {
  static {
    // Static initialization code
    MyClass.count = 0;
  }
}
```
