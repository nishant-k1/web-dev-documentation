# ES6 (2015) - Destructuring Assignment

Destructuring Assignment was introduced in ES6, allowing you to extract values from arrays or properties from objects into distinct variables.

## What Changed in ES6?

### Before ES6:

```javascript
// Had to manually extract values
const user = { name: "John", age: 30 };
const name = user.name;
const age = user.age;

// Array values
const arr = [1, 2, 3];
const first = arr[0];
const second = arr[1];
```

### After ES6:

```javascript
// Object destructuring
const { name, age } = user;

// Array destructuring
const [first, second, third] = arr;
```

---

## Key Features

- **Object Destructuring**: Extract properties by key name
- **Array Destructuring**: Extract values by position
- **Nested Destructuring**: Destructure nested structures
- **Default Values**: Provide defaults for missing values
- **Rest Pattern**: Collect remaining elements
- **Use in Parameters**: Destructure function parameters directly
