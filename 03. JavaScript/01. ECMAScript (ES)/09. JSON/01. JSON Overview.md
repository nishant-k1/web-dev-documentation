# JSON Overview

JSON (JavaScript Object Notation) is a lightweight data-interchange format. In JavaScript, JSON is provided as a built-in object with methods for serialization and deserialization.

## Basic Usage

```javascript
// Converting JavaScript to JSON string
const obj = { name: "John", age: 30 };
const jsonString = JSON.stringify(obj);
// '{"name":"John","age":30}'

// Converting JSON string to JavaScript
const parsed = JSON.parse(jsonString);
// { name: 'John', age: 30 }
```

## Key Methods

- **JSON.stringify()**: Converts JavaScript value to JSON string
- **JSON.parse()**: Converts JSON string to JavaScript value

## Important Notes

- **Circular References**: JSON.stringify() cannot handle circular references
- **Functions/Undefined**: Functions and undefined values are omitted
- **Date Objects**: Dates are converted to strings
- **Revivers/Replacers**: Can customize serialization/deserialization

## Advanced Features

```javascript
// With replacer function
JSON.stringify(obj, (key, value) => {
  // Custom transformation
});

// With replacer array (whitelist)
JSON.stringify(obj, ["name", "age"]);

// With space (pretty printing)
JSON.stringify(obj, null, 2);

// With reviver function
JSON.parse(jsonString, (key, value) => {
  // Custom transformation
});
```

## Common Use Cases

- API data serialization/deserialization
- Configuration files
- Data storage (localStorage, sessionStorage)
- Inter-service communication
- Logging and debugging
