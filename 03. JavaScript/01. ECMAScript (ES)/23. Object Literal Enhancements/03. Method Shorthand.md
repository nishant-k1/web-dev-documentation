# Method Shorthand

Method shorthand allows you to define methods in object literals without using the `function` keyword.

---

## Syntax

```javascript
// ❌ Before ES6
const obj = {
  greet: function () {
    return "Hello";
  },
  calculate: function (a, b) {
    return a + b;
  },
};

// ✅ ES6 - Method Shorthand
const obj = {
  greet() {
    return "Hello";
  },
  calculate(a, b) {
    return a + b;
  },
};
```

---

## Key Differences from Regular Functions

| Feature          | Method Shorthand             | Regular Function Property              |
| ---------------- | ---------------------------- | -------------------------------------- |
| Syntax           | `{ method() {} }`            | `{ method: function() {} }`            |
| `this` binding   | Bound to the object          | Bound to the object                    |
| Hoisting         | Not hoisted                  | Function expression (not hoisted)      |
| Can be generator | Yes: `{ *method() {} }`      | Yes: `{ method: function*() {} }`      |
| Can be async     | Yes: `{ async method() {} }` | Yes: `{ method: async function() {} }` |

## Examples

### Basic Method Shorthand

```javascript
const calculator = {
  add(a, b) {
    return a + b;
  },
  subtract(a, b) {
    return a - b;
  },
  multiply(a, b) {
    return a * b;
  },
  divide(a, b) {
    return b !== 0 ? a / b : null;
  },
};

console.log(calculator.add(5, 3)); // 8
console.log(calculator.multiply(4, 7)); // 28
```

### Method Shorthand with `this`

```javascript
const user = {
  firstName: "John",
  lastName: "Doe",

  // Method shorthand - 'this' is bound correctly
  getFullName() {
    return `${this.firstName} ${this.lastName}`;
  },

  // Regular function - 'this' is also bound correctly
  getInitials: function () {
    return `${this.firstName[0]}${this.lastName[0]}`;
  },
};

console.log(user.getFullName()); // "John Doe"
console.log(user.getInitials()); // "JD"
```

### Comparing Method Shorthand vs Arrow Functions

```javascript
const obj = {
  name: "Test",

  // ✅ Method shorthand - 'this' refers to obj
  regularMethod() {
    return this.name;
  },

  // ❌ Arrow function - 'this' refers to outer scope (or undefined in strict mode)
  arrowMethod: () => {
    return this.name; // 'this' is not bound to obj
  },
};

console.log(obj.regularMethod()); // "Test"
console.log(obj.arrowMethod()); // undefined (or error in strict mode)
```

### Async Methods

```javascript
const apiClient = {
  async fetchUser(id) {
    const response = await fetch(`/api/users/${id}`);
    return response.json();
  },

  async fetchPosts(userId) {
    const response = await fetch(`/api/users/${userId}/posts`);
    return response.json();
  },
};

const user = await apiClient.fetchUser(123);
```

### Generator Methods

```javascript
const counter = {
  *generateNumbers(start, end) {
    for (let i = start; i <= end; i++) {
      yield i;
    }
  },
};

const numbers = [...counter.generateNumbers(1, 5)];
console.log(numbers); // [1, 2, 3, 4, 5]
```

### Combined: Property Shorthand + Method Shorthand

```javascript
function createCounter(name) {
  let count = 0;

  return {
    name, // Property shorthand
    increment() {
      // Method shorthand
      return ++count;
    },
    decrement() {
      // Method shorthand
      return --count;
    },
    getCount() {
      // Method shorthand
      return count;
    },
  };
}

const counter = createCounter("My Counter");
console.log(counter.increment()); // 1
console.log(counter.name); // "My Counter"
```

## Common Interview Questions

### Q: What's the difference between method shorthand and arrow functions in objects?

```javascript
const obj = {
  name: "Test",

  // Method shorthand
  method1() {
    return this.name; // ✅ 'this' is obj
  },

  // Arrow function
  method2: () => {
    return this.name; // ❌ 'this' is not obj (likely undefined)
  },
};
```

**Answer:** Method shorthand binds `this` to the object, while arrow functions use lexical `this` from the enclosing scope.

## Best Practices

### ✅ DO: Use Method Shorthand for Object Methods

```javascript
const obj = {
  greet() {
    return "Hello";
  }, // ✅ Preferred
};
```

### ❌ DON'T: Use Arrow Functions When You Need `this` Binding

```javascript
const obj = {
  name: "Test",
  // ❌ Arrow function - 'this' won't work
  getName: () => this.name,

  // ✅ Method shorthand - 'this' works correctly
  getName() {
    return this.name;
  },
};
```
