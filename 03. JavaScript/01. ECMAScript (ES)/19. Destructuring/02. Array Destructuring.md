# Array Destructuring

Array destructuring allows you to extract values from arrays based on their **position** (index).

---

## Basic Syntax

```javascript
const arr = [1, 2, 3];
const [first, second, third] = arr;

console.log(first); // 1
console.log(second); // 2
console.log(third); // 3
```

---

## Key Characteristics

- **Position matters**: Values are extracted based on array index
- **Order matters**: Variables are assigned left-to-right
- **Can skip elements**: Use commas to skip positions
- **Can use rest**: Capture remaining elements

---

## Common Patterns

### 1. Basic Extraction

```javascript
const [a, b, c] = [1, 2, 3];
```

### 2. Skipping Elements

```javascript
const [first, , third] = [1, 2, 3];
console.log(first); // 1
console.log(third); // 3
// second is skipped
```

### 3. Default Values

```javascript
const [a = 10, b = 20] = [1];
console.log(a); // 1 (from array)
console.log(b); // 20 (default value)
```

### 4. Rest Operator

```javascript
const [first, ...rest] = [1, 2, 3, 4, 5];
console.log(first); // 1
console.log(rest); // [2, 3, 4, 5]
```

### 5. Swapping Variables

```javascript
let a = 1;
let b = 2;
[a, b] = [b, a];
console.log(a); // 2
console.log(b); // 1
```

### 6. Function Return Values

```javascript
function getValues() {
  return [1, 2, 3];
}

const [x, y, z] = getValues();
```

### 7. Nested Arrays

```javascript
const [a, [b, c]] = [1, [2, 3]];
console.log(a); // 1
console.log(b); // 2
console.log(c); // 3
```

---

## Use Cases

| Use Case                        | Example                                |
| ------------------------------- | -------------------------------------- |
| Swapping variables              | `[a, b] = [b, a]`                      |
| Accessing first few elements    | `const [first, second] = arr;`         |
| Ignoring elements               | `const [,, third] = arr;`              |
| Handling function return values | `const [err, result] = await fn();`    |
| Looping with `Object.entries()` | `for ([key, val] of Object.entries())` |

---

## When NOT to Use

- When array is very large and you need items from random/end positions
- When array size is dynamic or unknown
- When you need clarity with lots of element skipping
