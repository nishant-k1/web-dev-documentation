# Summary

### Operator Types

| Type    | Operand Count | Example                 |
| ------- | ------------- | ----------------------- |
| Unary   | 1             | `!x`, `++x`, `typeof x` |
| Binary  | 2             | `a + b`, `x ?? y`       |
| Ternary | 3             | `cond ? x : y`          |

### Key Concepts

1. **Short-circuiting**: Logical operators (`&&`, `||`) stop evaluating when the result is determined
2. **Type coercion**: Some operators (`==`, `+` with strings) perform type conversion
3. **Precedence**: Determines evaluation order (see [02. Operators Precedence](./02.%20Operators%20Precedence.md))
4. **Associativity**: Determines evaluation direction when precedence is equal
5. **Return values**: Logical operators return values, not just booleans

### Important Notes

- Spread and Rest (`...`) are **syntax**, not operators
- `++` and `--` are **unary**, not binary
- `!` is **unary**, not binary
- `~` is **unary**, not binary
- `?.` (Optional Chaining) is a **syntax construct**, not a traditional operator
