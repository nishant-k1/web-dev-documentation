# Destructuring Overview

Destructuring is a JavaScript syntax that allows you to extract values from arrays or properties from objects into distinct variables. It's a powerful feature that makes code more concise and readable.

---

## What is Destructuring?

**Destructuring** (officially called **Destructuring Assignment**) is the process of unpacking values from arrays or properties from objects into distinct variables.

### Key Principle

- **Object destructuring**: The **key** of values matters
- **Array destructuring**: The **position** of values matters

---

## Basic Syntax

### Object Destructuring

```javascript
const user = { name: "John", age: 30 };
const { name, age } = user;
console.log(name); // "John"
console.log(age); // 30
```

### Array Destructuring

```javascript
const numbers = [1, 2, 3];
const [first, second, third] = numbers;
console.log(first); // 1
console.log(second); // 2
console.log(third); // 3
```

---

## Contexts Where Destructuring Can Be Used

1. **Variable Assignment** - `const { a } = obj`
2. **Function Parameters** - `function fn({ a }) {}`
3. **Default Parameters** - `function fn({ a = 1 } = {}) {}`
4. **for...of Loops** - `for (const { a } of arr) {}`
5. **catch Clauses** - `catch ({ message }) {}`
6. **import Statements** - `import { x } from 'module'` (compile-time, not runtime)
