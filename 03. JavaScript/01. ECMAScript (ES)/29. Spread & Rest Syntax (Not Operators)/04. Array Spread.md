# Array Spread

Only enumerable own Elements are copied (not inherited ones)

## 1. Shallow Copying Arrays

```javascript
// Shallow copy
let original = [1, 2, 3];
let copy = [...original];
copy.push(4);
// original: [1, 2, 3]
// copy: [1, 2, 3, 4]
```

## 2. Immutably Adding Elements to Array

- Order Matters in Array Spread → affects where elements appear (position)
- spread copies Elements left to right.
- since keys (index) are always distinct so overriding not possible using spread syntax in Array, only adding takes place

```javascript
// Creates new array, doesn't mutate original
let items = [1, 2, 3];
let newItems = [...items, 4];
```

## 3. Merging Arrays

```javascript
let arr1 = [1, 2];
let arr2 = [3, 4];
let arr3 = [5, 6];
let merged = [...arr1, ...arr2, ...arr3];
// Result: [1, 2, 3, 4, 5, 6]
```

## Common Patterns and Use Cases of spread syntax in Arrays

## 1. Flattening Nested Arrays

1. Without Flat method

   ```js
   let arrays = [
     [1, 2],
     [3, 4],
     [5, 6],
   ];
   let flattened = [].concat(...arrays);
   ```

2. with flat() method it would have been done like below

   ```js
   let flattened2 = arrays.flat();
   // Result: [1, 2, 3, 4, 5, 6]
   ```

## 5. Function Arguments

```javascript
function sum(a, b, c) {
  return a + b + c;
}

let numbers = [1, 2, 3];
sum(...numbers); // 6
// Equivalent to: sum(1, 2, 3)
```

## 6. Math Functions

```javascript
let numbers = [5, 2, 8, 1, 9];
Math.max(...numbers); // 9
Math.min(...numbers); // 1

// Without spread (old way)
Math.max.apply(null, numbers); // 9
```
