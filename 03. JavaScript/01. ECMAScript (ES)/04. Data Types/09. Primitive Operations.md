### 4. **Comparing**

Comparing primitive values.

**Characteristics:**

- Primitives are compared by **value**
- Two primitives with the same value are equal
- Uses `===` or `==` operators

**Examples:**

```js
console.log(5 === 5); // true (same value)
console.log("hello" === "hello"); // true (same value)
```

### 5. **Copying**

Copying primitive values.

**Characteristics:**

- Copying creates a completely independent copy
- Changes to one variable do not affect others
- Primitives are copied by **value**

**Examples:**

```js
let x = 100;
let y = x; // y gets a copy of x
y = 200;
console.log(x); // 100 (x remains unaffected)
```
