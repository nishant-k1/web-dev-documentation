# Rest Syntax (`...`)

Rest syntax collects remaining values into a single variable which returns array in function parameters and array destructuring while an object in object destructuring.

1. **Function parameters** → collects multiple elements into an `array`
2. **Object destructuring** → collects remaining properties into an `object`
   In object destructuring, rest always produces an object (never undefined or null).
3. **Array destructuring** → collects remaining properties into an `array`
   Rest in array destructuring never becomes null or undefined. It always becomes an array.

## 1. Rest Parameters in Functions

### 1. Collecting Arguments

```javascript
// Collecting all arguments into an array
function sum(...numbers) {
  // Here numbers is an array
  return numbers.reduce((total, num) => total + num, 0);
}

sum(1, 2, 3, 4); // 10
sum(5, 10); // 15
```

### 2. Combining with Regular Parameters

```javascript
function greet(greeting, ...names) {
  return names.map((name) => `${greeting}, ${name}!`).join(" ");
}

greet("Hello", "John", "Jane", "Bob");
// Result: "Hello, John! Hello, Jane! Hello, Bob!"
```

Example 2

```js
function createUser(name, ...additionalInfo) {
  const [email, { age, city }] = additionalInfo;

  return {
    name,
    email,
    age,
    city,
  };
}

createUser("John", "john@example.com", { age: 30, city: "NYC" });

// Result: { name: "John", email: "john@example.com", age: 30, city: "NYC" }
```

### 3. Rest Must Be Last Parameter in all three cases that is function parameter, object destructuring and array destructuring

```javascript
// ✅ Correct
function example(a, b, ...rest) {
  // ...
}

// ❌ Syntax Error
function example(...rest, a) {
  // SyntaxError: Rest parameter must be last formal parameter
}
```

### 4. With Default Values

```javascript
let [first = 0, ...rest] = [];
// first: 0
// rest: []
```

### 5. Array Destructuring with Rest

1. **Collecting Remaining Elements**

   ```javascript
   let [first, second, ...rest] = [1, 2, 3, 4, 5];
   // first: 1
   // second: 2
   // rest: [3, 4, 5]
   ```

   ```js
   // Skipping elements
   let [first, , third, ...rest] = [1, 2, 3, 4, 5];
   // first: 1
   // third: 3
   // rest: [4, 5]
   ```

   ```javascript
   let [first, second, ...rest] = [];
   // first: undefined
   // second: undefined
   // rest: []
   ```

### 6. Object Destructuring with Rest

1. **Collecting Remaining Properties**

   ```javascript
   let { name, age, ...otherProps } = {
     name: "John",
     age: 30,
     city: "NYC",
     country: "USA",
   };
   // name: "John"
   // age: 30
   // otherProps: { city: "NYC", country: "USA" }
   ```

   ```javascript
   let { name, age, ...otherProps } = {};
   // name: undefined
   // age: undefined
   // otherProps: {}
   ```

2. **Removing Properties**

   ```javascript
   let user = { name: "John", age: 30, password: "secret" };
   let { password, ...publicUser } = user;
   // publicUser: { name: "John", age: 30 }
   // password is excluded
   ```

### 7: Converting Arguments Object

```js
// Old way
function oldWay() {
  let args = Array.prototype.slice.call(arguments);
  // ...
}

// Modern way with rest
function newWay(...args) {
  // args is already an array
  // ...
}
```

### 8: Removing Properties

```js
function removeProperty(obj, propToRemove) {
  const { [propToRemove]: removed, ...rest } = obj;
  return rest;
}

let user = { name: "John", age: 30, password: "secret" };
let safe = removeProperty(user, "password");
// Result: { name: "John", age: 30 }
```
