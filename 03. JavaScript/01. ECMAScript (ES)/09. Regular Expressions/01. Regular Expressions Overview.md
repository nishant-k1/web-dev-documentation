# Regular Expressions Overview

Regular Expressions (RegExp) are patterns used to match character combinations in strings. In JavaScript, regular expressions are also objects.

## Basic Usage

```javascript
// Literal notation
const regex = /pattern/flags;

// Constructor notation
const regex = new RegExp('pattern', 'flags');

// Methods
regex.test('string');      // Returns true/false
regex.exec('string');      // Returns match array or null
'string'.match(regex);     // Returns match array or null
'string'.replace(regex, replacement);
'string'.search(regex);    // Returns index or -1
```

## Key Concepts

- **Patterns**: Define what to match
- **Flags**: Modify matching behavior (g, i, m, s, u, y)
- **Character Classes**: Match sets of characters
- **Quantifiers**: Specify how many times a pattern should match
- **Groups and Capturing**: Extract parts of matches
- **Lookaheads/Lookbehinds**: Assert conditions without consuming characters

## Common Use Cases

- String validation (email, phone, etc.)
- String extraction and replacement
- Pattern matching and searching
- Input sanitization
- Parsing and data extraction
