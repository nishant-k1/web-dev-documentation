# Property Shorthand

Property shorthand allows you to omit the property value when the key matches a variable name.

---

## Syntax

```javascript
// ❌ Before ES6
const name = "John";
const age = 30;
const user = {
  name: name,
  age: age,
};

// ✅ ES6 - Property Shorthand
const name = "John";
const age = 30;
const user = {
  name, // Same as name: name
  age, // Same as age: age
};
```

---

## How It Works

When the property name and the variable name are the same, you can use the shorthand syntax. JavaScript automatically creates a property with the same name as the variable and assigns the variable's value to it.

---

## Examples

### Basic Usage

```javascript
function createUser(name, email) {
  return {
    name, // name: name
    email, // email: email
  };
}

const user = createUser("Alice", "alice@example.com");
console.log(user); // { name: "Alice", email: "alice@example.com" }
```

### Combining with Regular Properties

```javascript
const status = "active";
const role = "admin";

const user = {
  status, // Shorthand
  role, // Shorthand
  createdAt: new Date(), // Regular property
  id: Math.random(), // Regular property
};
```

### Returning Objects from Functions

```javascript
function makeUser(name, age, email) {
  return {
    name,
    age,
    email,
    isActive: true,
  };
}

const user = makeUser("Bob", 25, "bob@example.com");
```

### Object Destructuring Combined with Shorthand

```javascript
function processUser({ name, age, email }) {
  // Return new object with same properties
  return {
    name,
    age,
    email,
    processedAt: new Date(),
  };
}

const original = { name: "John", age: 30, email: "john@example.com" };
const processed = processUser(original);
```

## Real-World Use Cases

### Redux Action Creators

```javascript
const addTodo = (text) => ({
  type: "ADD_TODO",
  text,
  id: Date.now(),
  completed: false,
});
```

### Configuration Objects

```javascript
function createConfig(host, port, ssl) {
  return {
    host, // Property shorthand
    port, // Property shorthand
    ssl, // Property shorthand
  };
}
```

### React Props

```javascript
const user = {
  name: "John Doe",
  email: "john@example.com",
  avatar: "/avatar.jpg",
};

<UserCard {...user} />; // Spreads properties
```

---

## Common Mistakes

### ❌ Don't Use Shorthand When Names Don't Match

```javascript
const fullName = "John Doe";
const user = {
  name: fullName, // ✅ Must use full syntax
  // name  // ❌ Wrong - would create name: undefined
};
```
