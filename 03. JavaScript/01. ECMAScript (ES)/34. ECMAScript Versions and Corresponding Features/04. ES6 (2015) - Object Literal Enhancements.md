# ES6 (2015) - Object Literal Enhancements

ES6 introduced three major enhancements to object literal syntax:

1. **Property Shorthand**
2. **Method Shorthand**
3. **Computed Property Names**

## What Changed in ES6?

### Before ES6:

```javascript
// Had to write property names and values explicitly
const name = "John";
const user = {
  name: name,
  greet: function () {
    return "Hello";
  },
};

// Couldn't use variables as keys directly
const key = "age";
const obj = {};
obj[key] = 30; // Had to assign after creation
```

### After ES6:

```javascript
// Property shorthand
const name = "John";
const user = {
  name, // âœ… Shorthand
  greet() {
    // âœ… Method shorthand
    return "Hello";
  },
  [key]: 30, // âœ… Computed property name
};
```

---

## Features

- **Property Shorthand**: `{ name }` instead of `{ name: name }`
- **Method Shorthand**: `{ greet() {} }` instead of `{ greet: function() {} }`
- **Computed Property Names**: `{ [key]: value }` for dynamic keys
