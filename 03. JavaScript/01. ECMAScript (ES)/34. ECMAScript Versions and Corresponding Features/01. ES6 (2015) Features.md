## Quick Reference: Before vs After ES6

### Before ES6

```javascript
// Function-scoped variables
var x = 1;

// Function expressions
var fn = function () {};

// Long object syntax
var obj = {
  name: name,
  greet: function () {},
};

// No modules
// No classes
// No promises
// No destructuring
```

### After ES6

```javascript
// Block-scoped variables
let x = 1;
const y = 2;

// Arrow functions
const fn = () => {};

// Object literal enhancements
const obj = {
  name,
  greet() {},
};

// Modules, classes, promises, destructuring...
```
