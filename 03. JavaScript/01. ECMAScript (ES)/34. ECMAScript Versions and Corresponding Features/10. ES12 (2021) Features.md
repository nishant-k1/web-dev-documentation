## Major ES12 Features

### **Promise.any**

ðŸ“– See: [21. Handling Async Js - Promises](../21.%20Handling%20Async%20Js%20-%20Belongs%20to%20ECMAScript/03.%20Promises.md)

Returns the first fulfilled promise from an iterable, rejecting only if all promises are rejected.

```javascript
Promise.any([promise1, promise2, promise3])
  .then((first) => console.log("First fulfilled:", first))
  .catch((error) => console.log("All rejected:", error));
```

### **Numeric Separator**

ðŸ“– See: [04. Data Types - Primitives](../04.%20Data%20Types/01.%20Primitives.md)

Underscores can be used to separate digits in numbers for improved readability.

```javascript
const billion = 1_000_000_000;
const hex = 0xff_ec_de_5e;
```

---

### **String.prototype.replaceAll**

ðŸ“– See: [04. Data Types - Strings](../04.%20Data%20Types/15.%20String.md)

Replaces all occurrences of a pattern in a string (without needing global regex flag).

```javascript
"hello hello".replaceAll("hello", "hi"); // "hi hi"
```

---

### **Logical Assignment Operators**

ðŸ“– See: [08. Operators](../08.%20Operators/1.%20Operators.md)

- `&&=` Logical AND assignment
- `||=` Logical OR assignment
- `??=` Nullish coalescing assignment

```javascript
a &&= b; // a = a && b
a ||= b; // a = a || b
a ??= b; // a = a ?? b
```
