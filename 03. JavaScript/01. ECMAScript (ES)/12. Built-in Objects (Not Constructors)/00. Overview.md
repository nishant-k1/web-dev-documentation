# Built-in Objects (Not Constructors)

These are **built-in objects** that are **not** constructors: you never use `new` with them (e.g. no `new Math()`, no `new JSON()`). They only provide static properties and methods (e.g. `Math.PI`, `Math.random()`, `JSON.stringify()`, `JSON.parse()`). They do not have a `.prototype` used for instances.

See parent folder [10. Standard Built-in Objects](../10.%20Standard%20Built-in%20Objects/) for the full overview and study order.
