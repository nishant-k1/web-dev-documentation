# Comma Expression

Evaluates multiple expressions and returns the last one.

## Syntax

```js
(expression1, expression2, expression3)
```

## Examples

```js
let x = (1, 2, 3); // x = 3 (returns last value)

for (let i = 0, j = 10; i < 5; i++, j--) {
  // Multiple expressions in loop
}
```

## Important Notes

- **Evaluation order**: Expressions are evaluated left-to-right
- **Return value**: Returns the value of the last expression
- **Common use**: Often used in for loops to initialize or update multiple variables
- **Readability**: Can reduce code readability - use sparingly
