# Conditional (Ternary) Expressions

The only ternary operator in JavaScript - `condition ? valueIfTrue : valueIfFalse`.

## Syntax

```js
condition ? valueIfTrue : valueIfFalse
```

## Examples

```js
let result = age >= 18 ? "adult" : "minor";
let max = a > b ? a : b;
let message = user ? `Hello, ${user}` : "Hello, Guest";
```

## Nested Ternary

```js
let grade = score >= 90 ? "A" : score >= 80 ? "B" : "C";
```

## Important Notes

- **Ternary operator**: The only operator that takes three operands
- **Alternative to if-else**: Provides a concise way to write conditional expressions
- **Readability**: Can become hard to read when nested - use with caution
