# Await Expression

Used in async functions to wait for promises.

## Syntax

```js
await promise;
await expression;
```

## Examples

```js
async function fetchData() {
  let data = await fetch("/api/data");
  return data.json();
}
```

## Error Handling

```js
async function fetchData() {
  try {
    let data = await fetch("/api/data");
    return data.json();
  } catch (error) {
    console.error("Error:", error);
  }
}
```

## Multiple Awaits

```js
async function fetchMultiple() {
  const data1 = await fetch("/api/data1");
  const data2 = await fetch("/api/data2");
  return { data1, data2 };
}
```

## Parallel Execution

```js
async function fetchParallel() {
  const [data1, data2] = await Promise.all([
    fetch("/api/data1"),
    fetch("/api/data2")
  ]);
  return { data1, data2 };
}
```

## Important Notes

- **Async functions**: Can only be used inside `async` functions
- **Promise handling**: Waits for promise to resolve or reject
- **Error handling**: Rejected promises throw errors that can be caught with try-catch
- **Non-blocking**: Does not block the main thread, but pauses function execution
