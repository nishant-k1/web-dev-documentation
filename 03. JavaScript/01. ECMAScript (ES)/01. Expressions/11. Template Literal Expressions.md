# Template Literal Expressions

String interpolation using backticks (ES6).

## Basic Syntax

```js
let name = "John";
let greeting = `Hello, ${name}!`; // "Hello, John!"
```

## Multi-line Strings

```js
let multiLine = `Line 1
Line 2`; // Multi-line string
```

## Expression Interpolation

```js
let expression = `Result: ${5 + 3}`; // "Result: 8"
```

## Tagged Template Literals

```js
function tag(strings, ...values) {
  // Process template literal
}

let result = tag`Hello ${name}!`;
```

## Important Notes

- **Backticks**: Use backticks (`` ` ``) instead of quotes
- **Interpolation**: Use `${expression}` to embed expressions
- **Multi-line**: Template literals preserve line breaks
- **Tagged templates**: Advanced feature for processing template literals
