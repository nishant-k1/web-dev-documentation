# Void Expression

Evaluates expression and returns `undefined`.

## Syntax

```js
void expression;
```

## Examples

```js
void 0; // undefined
void (5 + 3); // undefined
void myFunction(); // undefined (function still executes)
```

## Common Use Cases

```js
// Creating undefined value
let x = void 0; // x = undefined

// Preventing navigation in links
<a href="javascript:void(0)">Click me</a>
```

## Important Notes

- **Return value**: Always returns `undefined`
- **Expression evaluation**: The expression is still evaluated (side effects occur)
- **Rarely used**: Modern JavaScript rarely uses `void` - prefer `undefined` directly
