# Increment and Decrement Expressions

Modify numeric values.

## Postfix Increment/Decrement

Returns value, then modifies:

```js
let x = 5;
x++; // Returns 5, then x becomes 6
x--; // Returns 6, then x becomes 5
```

## Prefix Increment/Decrement

Modifies, then returns value:

```js
let x = 5;
++x; // x becomes 6, returns 6
--x; // x becomes 5, returns 5
```

## Examples

```js
let a = 5;
let b = a++; // b = 5, a = 6

let c = 5;
let d = ++c; // c = 6, d = 6
```

## Important Notes

- **Postfix**: Returns the original value, then increments/decrements
- **Prefix**: Increments/decrements first, then returns the new value
- **Side effects**: These expressions modify the variable (side effect)
- **Common use**: Often used in loops and counters
