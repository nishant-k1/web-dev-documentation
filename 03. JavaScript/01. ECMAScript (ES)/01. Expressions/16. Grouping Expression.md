# Grouping Expression

Parentheses `()` change evaluation order and group expressions.

## Examples

```js
(2 + 3) * 4; // 20 (parentheses change order)
2 + 3 * 4; // 14 (multiplication first)
```

## Operator Precedence Override

```js
// Without parentheses (follows operator precedence)
2 + 3 * 4; // 14

// With parentheses (overrides precedence)
(2 + 3) * 4; // 20
```

## Function Arguments

```js
myFunction(1 + 2, 3 * 4); // Arguments are grouped expressions
```

## Important Notes

- **Precedence**: Parentheses override operator precedence
- **Grouping**: Used to group expressions and clarify evaluation order
- **Readability**: Can improve code readability by making evaluation order explicit
