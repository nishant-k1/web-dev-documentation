# Arithmetic Expressions

Perform mathematical operations.

## Addition

```js
3 + 4; // 7
"hello" + " world"; // "hello world" (string concatenation)
```

## Subtraction

```js
10 - 5; // 5
```

## Multiplication

```js
3 * 4; // 12
```

## Division

```js
10 / 2; // 5
```

## Modulo (Remainder)

```js
10 % 3; // 1
```

## Exponentiation

```js
2 ** 3; // 8
```

## Unary Plus

```js
+"5"; // 5 (converts to number)
```

## Unary Negation

```js
-5; // -5
```
