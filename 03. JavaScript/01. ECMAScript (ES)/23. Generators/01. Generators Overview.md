# Generators Overview

Generator functions are a special type of function that can be paused and resumed, allowing you to control the execution flow. They were introduced in ES6.

## Key Characteristics

- **Pausable Execution**: Can yield control back to the caller
- **Stateful**: Maintain state between pauses
- **Iterable**: Generators implement the iterator protocol
- **Lazy Evaluation**: Values are generated on-demand

## Basic Usage

```javascript
function* generatorFunction() {
  yield 1;
  yield 2;
  yield 3;
}

const gen = generatorFunction();
console.log(gen.next()); // { value: 1, done: false }
console.log(gen.next()); // { value: 2, done: false }
console.log(gen.next()); // { value: 3, done: true }
```

## Key Concepts

- **Generator Functions**: Functions declared with `function*`
- **Yield Expression**: Pauses execution and returns a value
- **Generator Objects**: Returned by generator functions
- **next() Method**: Resumes execution and receives yielded values
- **Async Generators**: Combine generators with async/await

## Common Use Cases

- Implementing custom iterators
- Creating infinite sequences
- Managing asynchronous flows
- Implementing coroutines
- Lazy evaluation and on-demand computation
