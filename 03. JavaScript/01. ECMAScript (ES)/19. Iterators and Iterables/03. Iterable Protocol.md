# Iterable Protocol

The **Iterable Protocol** defines how to make an object iterable. An iterable is an object that implements the `Symbol.iterator` method.

---

## Iterable Requirements

An object is iterable if it has a `Symbol.iterator` method that:

- Returns an iterator object
- The iterator implements the Iterator Protocol

---

## Basic Iterable

```javascript
const iterable = {
  data: [1, 2, 3],

  [Symbol.iterator]() {
    let index = 0;
    const data = this.data;

    return {
      next() {
        if (index < data.length) {
          return {
            value: data[index++],
            done: false,
          };
        }
        return { done: true };
      },
    };
  },
};

// Now can use with for...of
for (const value of iterable) {
  console.log(value); // 1, 2, 3
}
```

---

## Using Symbol.iterator

`Symbol.iterator` is a well-known symbol that defines the default iterator:

```javascript
const arr = [1, 2, 3];
const iterator = arr[Symbol.iterator]();

console.log(iterator.next()); // { value: 1, done: false }
console.log(iterator.next()); // { value: 2, done: false }
console.log(iterator.next()); // { value: 3, done: false }
console.log(iterator.next()); // { done: true }
```

---

## Built-in Iterables

These are iterable by default:

- **Arrays**: `[1, 2, 3]`
- **Strings**: `"hello"`
- **Maps**: `new Map()`
- **Sets**: `new Set()`
- **NodeLists**: `document.querySelectorAll()`
- **Arguments**: `arguments` object
- **TypedArrays**: `Uint8Array`, etc.

---

## Checking if Iterable

```javascript
function isIterable(obj) {
  return obj != null && typeof obj[Symbol.iterator] === "function";
}

console.log(isIterable([1, 2, 3])); // true
console.log(isIterable("hello")); // true
console.log(isIterable({})); // false
```
