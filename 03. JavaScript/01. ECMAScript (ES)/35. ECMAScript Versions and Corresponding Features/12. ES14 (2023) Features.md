## Major ES14 Features

### **Array.prototype.findLast and findLastIndex**

ðŸ“– See: [04. Data Types - Arrays](../04.%20Data%20Types/)

Find the last element or index of an element in an array that satisfies a test function.

```javascript
const arr = [1, 2, 3, 4, 3];
arr.findLast((x) => x === 3); // 3
arr.findLastIndex((x) => x === 3); // 4
```

### **Array.prototype.toSorted, toReversed, toSpliced, with**

ðŸ“– See: [04. Data Types - Arrays](../04.%20Data%20Types/)

New immutable array methods that return new arrays instead of mutating existing ones.

```javascript
const arr = [3, 1, 2];
arr.toSorted(); // [1, 2, 3] (doesn't mutate arr)
arr.toReversed(); // [2, 1, 3] (doesn't mutate arr)
arr.with(0, 4); // [4, 1, 2] (doesn't mutate arr)
```
