### **Promise.prototype.finally**

ğŸ“– See: [22. Handling Async Js - Promises](../21.%20Handling%20Async%20Js%20-%20Belongs%20to%20ECMAScript/03.%20Promises.md)

Executes code after a promise settles (fulfilled or rejected).

```javascript
promise
  .then((result) => {})
  .catch((error) => {})
  .finally(() => {
    // Always executes
  });
```
